# CLAUDE.md

Guidance for Claude Code (or Cursor) when working in this repository.

## Project
Capstone project for the FlyRank AI Internship, Frontend AI Engineering track. The goal is to build an AI-assisted frontend application while practicing an AI-augmented development workflow.

## Stack
Language: TypeScript. Framework: React (Vite) or Next.js. Styling: Tailwind CSS. Linting and formatting: ESLint and Prettier. Package manager: npm. Testing: Vitest and React Testing Library.

## Conventions
Commit messages follow Conventional Commits (feat, fix, docs, chore, refactor, test, style). Components are functional, written in TypeScript, and colocated with their styles and tests. Pull requests and commits stay small and focused. README.md is kept up to date as the project evolves.

## Working With AI Assistants
Explain the reasoning behind non-trivial changes in commit messages or PR descriptions. Review suggested changes for correctness and alignment with the conventions above before applying them. Use the assistant to review documentation, such as the README, and suggest concrete improvements before merging.
